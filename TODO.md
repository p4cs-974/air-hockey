# Requisitos:

-[X] Players na vertical (um na parte de cima e outro na parte de baixo)
-[X] IA básica controlando o player de baixo (Ghost System AI implementada)
-[X] Mouse controlando o player de cima (Player 1 - humano)
-[X] Os players podem se mover em qualquer direção, mas não podem invadir o campo adversário.
-[X] Imagem de fundo.
-[X] Palhetas redondas.
  - [X] Arrumar colisão
-[X] Tamanho do gol reduzido (não é a parede inteira) e com cores diferentes.
-[X] Sons de colisão

## Ghost System AI Features:

-[X] Ghost Puck simulation for trajectory prediction
-[X] Bounce Paradox Limiter (speed-based prediction depth)
-[X] State Machine (DEFEND, ATTACK, RECOVER)
-[X] Telegraph effect (velocity ramp-up during ATTACK)
-[X] Strike Personality (strategic return angles)
-[X] Movement Constraints (acceleration, overshoot)
-[X] Debug visualization (prediction point, state indicator)
